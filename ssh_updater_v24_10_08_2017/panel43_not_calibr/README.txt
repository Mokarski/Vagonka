Процедурв смены версии ПО для модулей PCProc и модулей 4.3%:

Шаг 1: Подключиться к устройству по его айпи адресу (PcProc - 192.168.1.121 Panel4,3 - 192.168.1.122)
		Для этого необходимо установить и настроить WinSCP или FileZilla для работы по протоколу sftp
		логин root, поле пароль оставить пустым.

Шаг 2: удалить все содержимое папки /mnt/software/bin

Шаг 3: Загрузить новую версию файлов ПО в папку /mnt/software/bin

Шаг 4: с помощью программы PUTTY подключиться к устройству (используя его IP адрес и протокол SSH)

Шаг 5: Сразу после подключения, на приглашение системы ввести root и нажать клавишу ентер
		далее вводим  следующее:
		cd пробел /mnt/software/bin снова клавишу ентер
		далее вводим:
		chmod пробел 775 пробел *  снова клавишу ентер
		и последнее что необходимо ввести:
		sync пробел && пробел reboot
		После чего устройство перезагрузиться
